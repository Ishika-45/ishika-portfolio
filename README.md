# Ishika Portfolio

A modern developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

## About

This portfolio showcases my projects, technical skills, and experience as a Full-Stack Developer. It is being built with a focus on performance, responsiveness, accessibility, and modern UI/UX practices.

## Tech Stack

* Next.js
* TypeScript
* React
* Tailwind CSS

## Features (Planned)

* Responsive Design
* Modern User Interface
* Project Showcase
* Skills Section
* Experience Timeline
* Contact Section
* Resume Download
* Performance Optimizations
* SEO Enhancements

## Project Status

🚧 Currently under development.

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the project locally.
