# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Ishika-45/ishika-portfolio.git
```

Move into the project

```bash
cd ishika-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
